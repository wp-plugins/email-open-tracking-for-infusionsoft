This plugin allows you to create automation based on email opens in Infusionsoft.

== Description ==

This plugin applies a tag in your Infusionsoft account when an email is opened.

== Installation ==

To Install:

1. Upload `isoft-opens` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the settings (instructions are included in the app)

== Frequently Asked Questions ==

= Will this plugin work out of the box? =

No - you will need to create a tag in Infusionsoft at the very least, and to actually succeed should build out a campaign for tracking.

= Where can I find documentation on the Infusionsoft setup required? =

Check out [Automate.fm/email-open-tracking-plugin](http://automate.fm/email-open-tracking-plugin)
